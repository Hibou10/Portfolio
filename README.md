## 🚀 Projekte

### 🦠 Covid-19 Datenanalyse
Aufbau einer vollständigen Analytics-Umgebung mit SQL Server als Datenquelle und Evidence.dev als Reporting-Framework. Entwicklung von SQL-Abfragen zur Datenaufbereitung und Erstellung interaktiver, codebasierter Dashboards.

`ETL` `SQLMesh` `DuckDB` `SQL Server` `Evidence.dev`

[📂 GitHub Repo](https://github.com/Hibou10/covid_dashboard) · [📊 Live Dashboard](https://covid-data-insights.netlify.app/)

---

### 🛒 Customer Shopping Analyse
Analyse des Einkaufsverhaltens von Kunden mithilfe von Python, SQL, Power BI und Streamlit. Explorative Datenanalyse (EDA) zur Identifikation von Kundensegmenten, Umsatztrends, Kaufverhalten und der Performance von Produktkategorien.

`Python` `Pandas` `PostgreSQL` `Power BI` `Streamlit`

[📂 GitHub Repo](https://github.com/Hibou10/customer-shopping-analysis) · [📊 Live Dashboard](https://customer-shopping.streamlit.app/)

---

### 🎬 Netflix Datenanalyse
Analyse eines Netflix-Datensatzes mit Fokus auf Datenqualität und explorative Datenanalyse (EDA). Bereinigung von Duplikaten, Identifikation fehlender Werte sowie Untersuchung der Datenstruktur mit Pandas und Seaborn.

`Pandas` `Seaborn` `EDA`

[📂 GitHub Repo](https://github.com/Hibou10/Netflix-Project)
