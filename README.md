# Persönliches Data-Portfolio

## Projektübersicht

Dieses Repository enthält mein persönliches Portfolio im Bereich  
**Data Analytics, Data Engineering und Business Intelligence**.

Die Website präsentiert meine Datenprojekte, technischen Kenntnisse und
eingesetzten Technologien. Mein Schwerpunkt liegt auf der Aufbereitung,
Analyse und Visualisierung von Daten sowie der Entwicklung interaktiver
Dashboards.

## Ausgewählte Projekte

### COVID-19-Datenanalyse

Aufbau einer vollständigen Analytics-Umgebung mit SQL Server als Datenquelle
und Evidence.dev als Reporting-Framework.

**Schwerpunkte:**

- Entwicklung von SQL-Abfragen zur Datenaufbereitung
- Aufbau von ETL- und Transformationsprozessen
- Entwicklung interaktiver, codebasierter Dashboards
- Visualisierung globaler COVID-19-Daten

**Technologien:**  
`SQL` `SQL Server` `SQLMesh` `DuckDB` `Evidence.dev`

🔗 [Live-Dashboard ansehen](https://covid-data-insights.netlify.app/)

---

### Customer Shopping Analysis

End-to-End-Datenprojekt zur Untersuchung des Einkaufsverhaltens von Kundinnen
und Kunden.

**Schwerpunkte:**

- Datenbereinigung und explorative Datenanalyse
- Analyse von Kundensegmenten und Umsatztrends
- SQL-Abfragen mit PostgreSQL
- Entwicklung eines Power-BI-Dashboards
- Entwicklung einer interaktiven Streamlit-Anwendung

**Technologien:**  
`Python` `Pandas` `PostgreSQL` `Power BI` `Streamlit`

🔗 [Live-Dashboard ansehen](https://customer-shopping.streamlit.app/)

---

### Netflix-Datenanalyse

Explorative Analyse eines Netflix-Datensatzes mit Fokus auf Datenqualität,
Bereinigung und Visualisierung.

**Schwerpunkte:**

- Identifikation und Entfernung von Duplikaten
- Analyse fehlender Werte
- Untersuchung der Datenstruktur
- Erstellung statistischer Visualisierungen

**Technologien:**  
`Python` `Pandas` `Seaborn` `EDA`
