# Persönliches Data-Portfolio

## Projektübersicht

Dieses Repository enthält mein persönliches Portfolio im Bereich  
**Data Analytics, Data Engineering und Business Intelligence**.

Die Website präsentiert meine Datenprojekte, technischen Kenntnisse und
eingesetzten Technologien. Mein Schwerpunkt liegt auf der Aufbereitung,
Analyse und Visualisierung von Daten sowie der Entwicklung interaktiver
Dashboards.
